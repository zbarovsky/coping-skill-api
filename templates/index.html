Welcome to the Coping Skills API

This relatively bare bones API lets you querery a list of 100 predetermined Coping Skills

Routes

As it currently stands, there are two RESTful routes you can utilize

  • 1: /api/copingskill/get_all will return you all current coping skills in the list
  • 2: /api/copingskill/:id will return you a specific coping skill from 0-99
